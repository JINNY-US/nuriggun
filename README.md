
# 📖서비스 명 : 누리꾼

우연히 뉴스를 보다가 “누리꾼의 반응이 뜨겁습니다” 같은 기자의 말에서 영감을 받아 누리꾼들의 진짜 반응이 어떤지, 기자들은 어디서 정보를 얻는 지 궁금했습니다.

유저가 기사를 내 다른 사람들의 진짜 반응을 확인 할 수 있고 공유할 수 있는 서비스를 생각하며 “ 유저가 기자가 되어 기사를 쓸 수 있는 sns 형 인터넷 신문사” 라는 주제로 프로젝트를 시작하게 되었습니다.
